<D2R d2s 파일 백업 프로그램 설명서>

1. "File Select" 버튼을 눌러 d2s 파일을 선택한다.
2. "Backup" 버튼을 누른다.
3. 하단에 백업 경로와 백업압축 파일이름을 확인 (백업경로 폴더로 바로이동 가능)

* 프로그램 중복 실행 방지 기능 걸려 있음. (만약 안켜진다면 켜져있는지 확인)
* 해당 d2s 파일이 있는 폴더에 D2R_backup 폴더가 생성
* d2s / 맵정보 / 기타 등등 캐릭터 연결 파일들이 있으면 같이 압축
* "캐릭터이름_날짜_시간_밀리초.zip"로 생성되어 압축파일이 중복 되지 않음.
